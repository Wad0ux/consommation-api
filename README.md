# Consommation API

Cette API permet d'ajouter et de récupérer les consommations de produits par client.

## Endpoints

- `POST /add` : Ajoute une consommation
- `GET /all` : Récupère toutes les consommations

## Déploiement sur Render

1. Poussez ce dossier sur GitHub
2. Allez sur [https://render.com](https://render.com)
3. Cliquez sur **"New Web Service"** et connectez le repo
4. Build command : `npm install`
5. Start command : `npm start`
